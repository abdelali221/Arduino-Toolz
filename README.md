# Arduino-Toolz
 A Toolbox for the Arduino!

# Features :

 - Clock
 - Terminal (Beta)
 - Analog/Digital PIN Tools (Beta)
 - Module Testers (IR/UltraSonic/DHT11/Keypad)
 - EEPROM Tool

 # How to Use :

Just upload the Project to your Arduino after downloading the required Libraries, then you'll be done!
